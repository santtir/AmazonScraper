#  Amazon Product Scraper

Este proyecto permite buscar productos en Amazon ingresando una palabra clave, y muestra los resultados del primer página (título, imagen, rating y cantidad de reviews).

---

##  Tecnologías usadas

- **Backend:** [Bun](https://bun.sh/) + Express + Axios + JSDOM
- **Frontend:** HTML, CSS, JavaScript (con Vite) + Bootstrap 5

---
